This was the final group project for my dynamic web development class. We did not have an assignment but rather a broad set of requirements such as using two APIs and connecting the code to a Supabase database we created. With these requirements, we decided to create a sort of book reccommender/database. It utilizes the Gutundex and Open Library APIs to find and display information (title, author, cover) about the most popular books both generally and based off of a specified genre. The site requires the CORS extension to run the fetches but the code can still be viewed from GitHub.

  • Languages: HTML, CSS, JavaScript
  • Skills used: APIs, REST, Database integration
